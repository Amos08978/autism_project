此簽署書簽署流程是為了保障您和孩子的個人資料安全，本系統遵循《個人資料保護法》相關規範。
讓孩子能使用本情緒學習系統並請先完成註冊與學習記錄(歷程紀錄)，並用於學習成效分析與系統優化提供數據參考， 提供本情緒學習系統之帳號註冊、學習記錄與成效分析，以及參與訪談的贈品寄送， 我們依《個人資料保護法》告知以下事項：

一、蒐集目的
  • 帳號註冊、身份確認、登入管理。
  • 學習評測與歷程紀錄、成效分析與報告呈現(以系統資料為主)。
  • 以去識別化方式做統計分析，用於研究、教學改進與系統優化。
  • 訪談聯繫與贈品寄送(地址 / Line 帳號僅限您選擇提供時)。
二、蒐集資料
  • 基本資料：孩子之姓名 / 代號 / 暱稱、年齡或年齡區間、年級、性別。
  • 聯絡資料(大人)：家長 / 老師 / 治療師之姓名(或稱謂)、Email、電話 / 手機(用於通知 / 找回帳號)。
  • 學習與評量資料(系統核心)：測驗結果、作答紀錄、情緒辨識正確率、任務完成狀況、練習頻率、反應時間等。
  • 系統紀錄：登入時間、裝置 / 瀏覽資訊、IP、Cookie(用於資安與系統穩定)。
  • 不會要求填寫身分證字號(AD)；若未來因研究設計或單位規範需要，會另外明確告知並取得同意。
三、使用方式、期間與對象
  • 使用方式：以人工或自動化方式蒐集、保存、處理與利用(如建檔、查詢、統計分析、通知聯繫)。
  • 保存期間：自您開始使用起至蒐集目的完成或您依法提出停止 / 刪除為止，若依法令或研究管理需要保存期限者，依規定辦理。
  • 利用地區：以台灣地區為主；若使用雲端服務涉及跨境傳輸，將採取適當保護措施。
  • 利用對象：本系統營運 / 研究團隊之授權人員及提供必要服務之合作廠商(如雲端主機、Email/簡訊服務、物流寄送)，僅在達成目的必要範圍內使用。
四、資料保護措施
  • 我們採取合理安全措施，例如：加密傳輸、存取權限控管、操作紀錄與內部管理機制，避免未經授權的存取或洩漏。
